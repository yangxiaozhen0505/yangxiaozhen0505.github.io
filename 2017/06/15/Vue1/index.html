Vue简介

  • 是一套轻量级的构建用户界面的渐进式框架
  • 定界符
  • 指令 v-xxx

特点

  • 它的核心只关注视图层
  • 易学、灵活
  • 渐进式框架

渐进式框架

  • 声明式渲染（无需关心怎么实现）
  • 组件系统 component
  • 客户端路由 vue-router
  • 大规模状态管理
  • 构建工具

Vue 两个核心

  • 响应的数据变化
    • 当数据发生改变，视图自动更新
  • 组合的视图组件
    • UI界面映射为组件树
    • 划分组件 可维护、可复用、可测试

      MVVM模式

  • M Model数据模型
  • V View 视图模板

  • VM view-model 视图模型

  • 将数据绑定到DOM上，监听视图的变化更改数据

Vue的安装

  • npm info vue 查看所有版本
  • npm init -y 初始化一个package.json文件。配置安装清单
  • npm install vue –save 安装项目依赖，指定版本vue@xx 不指定版本号，安装的是最新版本

指令

  • 特殊自定义行间属性
    • v-model 双向数据绑定 只能用于表单元素
    • v-for 循环顺序 value key
    • v- bind 绑定动态属性 可以简写为:
    • v-on 可以简写为 @ 后边跟事件
    • v-if/show 值为布尔类型，true就显示，false则为隐藏 频繁显示隐藏，一般用show
